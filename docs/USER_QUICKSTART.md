# USER QUICKSTART - CoastalRisk WebApp v1

## Dùng nhanh
1. Mở frontend `frontend/index.html`
2. A1: Upload file khảo sát chuyên gia
3. A2: Upload file dataset công trình
4. A3: Nhập vùng và bấm chạy huấn luyện
5. A4: Tải danh sách model và đặt model mặc định nếu cần
6. B2: Nhập X1..X6, chọn model, chạy dự báo
7. B3: Xem lịch sử, xem payload và tải báo cáo Word

## Ghi chú
- Nếu vừa train xong, nên sang A4 hoặc B2 để xác nhận model đã xuất hiện.
- Nếu muốn tải Word, hãy bấm `Xem` một history trước, rồi bấm `Tải Word`.
